Система распределённого наблюдения с машинным зрением для защиты от кошачьих вторжений 

**Кросплатформенная система наблюдения**, объединяющая веб-камеры, мобильные устройства 
и планируемую робототехнику в единую сеть с интерфейсом в локальной сети.

## 🔹 **Основные возможности**
✅ **Мультиплатформенный сбор данных**
- Использование веб-камер (десктоп) и старых телефонов (Android как камеры).

✅ **Машинное зрение для анализа**
- Обнаружение движений/объектов с помощью **YOLOv4-tiny** (или других моделей).
- Распознавание категорий ("кошка").

✅ **Коммуникация между устройствами**
- Обмен данными между камерами через локальную сеть.

✅ **Будущая интеграция робототехники**
- Поддержка робота-патруля.
- Автоматическое реагирование на угрозы (например, звуковой сигнал или приближение робота собаки к кошке исходя из ее позиции на камере).

## 🛠 **Технический стек**
| Компонент             | Технология  
| **Основной фреймворк**| C++ |
| **Машинное обучение** | YOLOv4-tiny, OpenCV               |
| **Локальная сеть**    | WebSockets, REST API (для обмена данными) |

## 🖥 **Архитектура системы**
```mermaid
graph TD
    A[Локальная сеть] --> B[Веб-камеры/Мобильные телефоны]
    B --> C[Программа 1: Анализ кадра 1 (YOLO)]
    B --> D[Программа 2: Анализ кадра 2 (YOLO)]

    D --> E

    F --> G[Звук + Визуальная индикация]
